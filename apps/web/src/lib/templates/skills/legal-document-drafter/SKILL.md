# 한국 법률문서 자동 작성 (Korean Legal Document Drafter)

118종 한국 법률문서 템플릿 기반의 자동 작성 스킬.
사용자 상황을 파악 → 적합한 문서 유형 추천 → 단계별 질문으로 정보 수집 → 완성된 법률문서 출력.

## 중요 안내사항

> **본 스킬은 참고용이며 법률 자문이 아닙니다.**
> 작성된 문서는 반드시 법률 전문가의 검토를 받은 후 사용하시기 바랍니다.
> 중요한 법적 분쟁이나 고액 거래의 경우 변호사 상담을 강력히 권장합니다.

## 문서 작성 워크플로우

```
1. 상황 파악 → 어떤 상황인지 이해
2. 문서 추천 → 적합한 문서 유형 제안
3. 정보 수집 → 단계별 질문 (한 번에 2-3개씩)
4. 문서 생성 → 완성된 법률문서 출력
5. 검토 안내 → 전문가 검토 권장
```

### 정보 수집 원칙
- **한 번에 2-3개 질문만** — 사용자를 압도하지 않기
- **선택지 제공** — 가능한 경우 번호 선택으로 간편화
- **예시 포함** — 어려운 항목은 구체적 예시 제공
- **순서대로** — 템플릿의 step 순서를 따르되, 자연스러운 대화 흐름 유지

## 문서 유형별 상세 템플릿

### 내용증명 공통 구조

내용증명은 우체국을 통해 상대방에게 법적 효력이 있는 의사를 전달하는 문서이다.

**공통 단계:**
1. **발신인/수신인 유형** — 법인 | 개인사업자 | 개인
2. **사건 내용** — 거래/계약 경위, 금액, 날짜
3. **청구 내용** — 요구 사항, 기한
4. **특별 손해** — 추가 손해 여부 (선택)
5. **발신인 상세정보** — 이름, 주소, 연락처, 주민등록번호
6. **수신인 상세정보** — 이름, 주소, 연락처

**내용증명 출력 형식:**
```
내 용 증 명

발신인: [이름]
        [주소]
        [연락처]

수신인: [이름]
        [주소]

제목: [청구 제목]

1. 발신인은 [날짜]에 수신인에게 [금액]원을 [사유]하였습니다.

2. 그러나 수신인은 [기한]까지 [의무내용]을 이행하지 않고 있습니다.

3. 이에 발신인은 수신인에게 본 내용증명을 통하여 [요구사항]을
   통지하오니, [기한]까지 [이행내용]하여 주시기 바랍니다.

4. 만일 위 기한 내에 이행하지 않을 경우, 발신인은 부득이
   법적 조치(지급명령, 민사소송 등)를 취할 것임을 알려드립니다.

[날짜]
발신인: [이름] (서명 또는 인)
```

### 지급명령 신청서 공통 구조

지급명령은 간이채권추심 절차로, 법원에 서면으로 청구하여 상대방에게 지급을 명하는 제도이다.

**공통 단계:**
1. **제출 방법** — 전자소송 | 직접제출
2. **채권자/채무자 유형** — 법인 | 개인
3. **청구 원인** — 대여/용역/보증금/임금 등 구체적 내용
4. **청구 금액** — 원금, 이자, 지연손해금
5. **입증방법** — 계약서, 이체확인증, 카톡내역 등
6. **관할 법원** — 채무자 주소지 기준
7. **채권자 상세정보** — 이름, 주민등록번호, 주소, 연락처
8. **채무자 상세정보** — 이름, 주민등록번호, 주소

**지급명령 신청서 출력 형식:**
```
지 급 명 령 신 청 서

채 권 자  [이름] ([주민등록번호])
          [주소]
          [연락처]

채 무 자  [이름] ([주민등록번호])
          [주소]

청 구 취 지

채무자는 채권자에게 금 [금액]원 및 이에 대하여 [기산일]부터
이 사건 지급명령 정본 송달일까지는 연 [약정이율]%의,
그 다음 날부터 다 갚는 날까지는 연 12%의 각 비율로 계산한
돈을 지급하라.

라는 지급명령을 구합니다.

청 구 원 인

1. 채권자는 [날짜]에 채무자에게 금 [금액]원을 [사유]하였습니다.

2. [변제기/기한 관련 내용]

3. 그러나 채무자는 현재까지 위 금원을 변제하지 아니하고 있습니다.

입 증 방 법

1. [증거서류 목록]

첨 부 서 류

1. 위 입증방법 각 1통
2. 송달료 납부서 1통

[날짜]

위 채권자 [이름] (서명 또는 인)

[법원명] 귀중
```

### 근로계약서 공통 구조

**공통 단계:**
1. **사용자/근로자 정보** — 법인 | 개인사업자 | 개인
2. **고용형태** — 정규직 | 기간제 | 단시간
3. **근무시간** — 주 40시간 | 직접입력, 고정OT 여부
4. **임금** — 연봉/월급, 구성항목(기본급, 식대, 상여금 등)
5. **근무내용** — 업무, 소속팀, 직급, 근무장소
6. **휴가/복리후생** — 연차, 퇴직금, 4대보험
7. **특약사항** — 수습기간, 비밀유지, 경업금지 등
8. **사용자 상세정보** — 사업장명, 대표자, 사업자번호, 주소
9. **근로자 상세정보** — 이름, 주민등록번호, 주소

**근로계약서 출력 형식:**
```
근 로 계 약 서

[사업장명] (이하 "갑"이라 한다)과(와) [근로자명] (이하 "을"이라 한다)은(는)
다음과 같이 근로계약을 체결한다.

제1조 (근로계약기간)
  [정규직: 기간의 정함이 없음 / 기간제: 20__년 __월 __일 ~ 20__년 __월 __일]

제2조 (근무장소)
  [근무장소]

제3조 (업무의 내용)
  [업무내용]

제4조 (근로시간)
  시업시각: [시간], 종업시각: [시간]
```

## 문서 추천 가이드

사용자의 상황 설명에서 적절한 문서를 추천하는 기준:

### "돈을 못 받았어요" 계열
- 빌려준 돈 → **내용증명(대여금 청구)** → 효과 없으면 **지급명령(대여금 청구)**
- 용역비/대금 → **내용증명(용역대금/물품대금 청구)** → **지급명령(용역/물품대금)**
- 보증금 → **내용증명(보증금 반환 청구)** → **지급명령(보증금 반환)**
- 월급/퇴직금 → **내용증명(급여, 퇴직금 청구)** → **지급명령(임금청구)**
- 임대료 → **내용증명(임대료 연체 해지/명도)** → **지급명령(임대료, 손해배상)**

### "계약 문제" 계열
- 계약 안 지켜요 → **내용증명(계약이행 청구)** 또는 **내용증명(위약금 청구)**
- 계약 해지하고 싶어요 → **내용증명(계약해지 통지)**
- 부동산 나가려는데 → **내용증명(부동산 계약해지-세입자용)**
- 세입자 내보내려면 → **내용증명(부동산 계약해지-임대인용)**

### "피해를 입었어요" 계열
- 폭행 → **합의서(폭행)** 또는 **고소장** 선택
- 온라인 모욕 → **고소장(모욕죄-온라인게임용)**
- 직장 성희롱 → **고소장(모욕죄-성희롱)**
- 중고거래 사기 → **고소장(사기죄-중고거래)**
- 교통사고 → **교통사고 합의서** (경미/민사/형사 구분)
- 명예훼손 → **합의서(명예훼손)**
- 손해배상 → **내용증명(손해배상 청구)**

### "직원 채용/퇴사" 계열
- 정규직 채용 → **근로계약서(정규직, 기간제)**
- 5인 미만 사업장 → **근로계약서(5인 미만)**
- 알바 채용 → **알바계약서(5인 이상)** 또는 **(편의점용, 5인 미만)**
- 연봉 갱신 → **근로계약서(연봉갱신용)**
- 직원 해고 → **해고예고통지서**
- 입사 서약 → **입사자서약서**
- 퇴사 처리 → **퇴사자서약서** + **경력증명서**

### "사업/투자" 계열
- NDA → **비밀유지계약서(NDA)** (일반/M&A/공동연구 구분)
- 동업 → **동업계약서(법인 미설립)** 또는 **주주간계약서**
- 투자 유치 → **투자계약서(SAFE)** 또는 **스톡옵션계약서**
- 주식 거래 → **주식양수도계약서**
- 용역 외주 → **용역계약서** (일반/IT/건설/광고/컨설팅 구분)
- 프랜차이즈 → **프랜차이즈 계약서**
- 사업 양수도 → **사업양수도 계약서**

### "서비스 운영" 계열
- 이용약관 필요 → **서비스이용약관** (일반/플랫폼/자동차/전자금융 구분)
- 개인정보처리방침 → **개인정보처리방침**
- 개인정보 동의서 → **개인정보처리동의서** 또는 **수집/이용동의서**

### "부동산" 계열
- 임대차 → **부동산 임대차 계약서(직거래용)**
- 상가 임대 → **상가임대차계약서**
- 매매 → **부동산 매매계약서**
- 가계약 → **부동산 가계약서**

## 문서 작성 시 주의사항

### 법적 유효성
1. **주민등록번호**: 실제 문서에서는 반드시 기재, 작성 시에는 "XXX-XXXXXXX" 같은 placeholder 사용 안내
2. **서명/날인**: 모든 당사자의 서명 또는 날인 필요 안내
3. **날짜**: 작성일 반드시 기재
4. **2부 이상**: 당사자 수만큼 작성하여 각 1부씩 보관 안내

### 내용증명 특별 주의
- 우체국 방문 발송 (등기 + 내용증명)
- 발송 전 사본 3부 준비 (발신인용, 수신인용, 우체국보관용)
- 배달증명 추가 신청 권장

### 지급명령 특별 주의
- 관할법원: 채무자 주소지 관할 법원
- 인지대: 청구금액에 따라 차등
- 송달료: 당사자 수 x 5,200원 x 3회분
- 채무자 이의 시 민사소송으로 이행

### 근로계약서 특별 주의
- 5인 이상/미만 사업장에 따라 적용 법조항 다름
- 최저임금 위반 여부 반드시 확인
- 4대보험 가입 의무 안내
- 근로계약서 교부 의무 (근로기준법 제17조)

7. **마크다운 code block** 안에 문서 본문 출력 (복사 편의)
