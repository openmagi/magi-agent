# Humanize Korean — AI 한글 티 제거 (봇 전용 v1.5)

사용자가 AI 티 제거/윤문을 요청하면 이 스킬을 발동한다. 한 응답 안에서 탐지→윤문→자체검증을 일괄 수행.

## 4대 철칙

1. **의미 불변** — 사실·주장·수치·고유명사·직접 인용은 100% 원문 보존.
2. **근거 기반** — 아래 패턴 표에 매핑되는 구간에만 수술적 수정. 탐지 없는 곳은 건드리지 않음.
3. **장르 유지** — 칼럼을 문학으로, 리포트를 에세이로 옮기지 않음.
4. **과윤문 금지** — 변경률 30% 초과 시 경고, 50% 초과 시 강제 중단.

## Do-NOT (탐지·윤문 모두 제외)

고유명사·제품명·모델명·기관명, 수치·날짜·단위, 큰따옴표 안 직접 인용, 법률 조문, 수학·화학·통계 표기, 영어 약어(LLM·GPU·MCP·API 등 업계 표준).

## 작업 순서

### 1. 입력 확인
- 사용자가 붙여넣은 텍스트 또는 "이전 응답 윤문해줘" 등 대상 확인
- 첫 300자로 장르 추정 (칼럼/리포트/블로그/공적/대화)

### 2. 패턴 탐지
- 위 표 기준으로 S1·S2 패턴 스캔
- Do-NOT list 엄격 적용 (고유명사·수치·인용 제외)
- 카테고리별 탐지 건수 집계

### 3. 수술적 윤문
- D(관용구 삭제) 먼저 → A → I → G → H → F → B → C·J → E 순
- 문단 단위 처리, 변경률 모니터링 (50% 임박 시 중단)
- 탐지되지 않은 구간은 원문 그대로 유지

### 4. 자체검증 6항
1. 고유명사·수치·날짜·인용 100% 보존
2. 변경률 30% 이하
3. 장르 이탈 없음
4. register 보존 (격식체→격식체)
5. S1 잔존 0건
6. 인공 표현 추가 없음

위반 항목 발견 시 해당 부분 롤백 후 1회 재시도.

### 5. 결과 전달

```
[윤문 완료] 변경률 X% / 등급 Y / 자체검증 N/6

---
(윤문본 전문)
---

**탐지 요약:** A:N건→M건 | D:N건→M건 | H:N건→M건 | ...
**주요 변경:** (before→after 하이라이트 3~5건)
```

## 등급 기준

- **A**: S1 잔존 0, S2 ≤2, 변경률 10~25%, 자체검증 6항 통과
- **B**: S1 잔존 0, S2 ≤4, 자체검증 5항+ 통과
- **C**: S1 1~2건 — "더 손봐줘"로 2차 윤문 가능
- **D**: S1 3건+ — 수동 검토 권고

## 후속 명령 대응

| 사용자 | 처리 |
|---|---|
| "특정 카테고리만 다시" | 해당 카테고리만 재탐지·재윤문 |

## 주의

- 이 스킬은 **문체 윤문 전용**. 내용 추가·삭제·번역·새 글 작성은 별도 스킬.
- 맞춤법·오탈자 교정은 윤문과 별개로 직접 처리.
- 영어 텍스트에는 발동하지 않음 (한글 전용).
